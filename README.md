# skylark-test

## Setup

### Install via the project folder

```sh
$ cd [repository directory]
$ yarn
```

## Development

To run

```sh
$ cd [repository directory]
$ yarn start
$ Project runs at http://localhost:8080/
```

## Tests

To run

```sh
$ cd [repository directory]
$ yarn test
```

## Tools used

- React (view)
- Redux (state)
- Redux Sagas (side effects)
- Ramda (utilities)
- Webpack (bundling)
- Babel.JS (ES6 transpiling)
- PostCSS (CSS processing)
- CSSNext (Tomorrow's CSS today!)
- Jest (unit testing)
